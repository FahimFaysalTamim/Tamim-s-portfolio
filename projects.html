03. Mini VSFS — Virtual Simple File System

Stack: C, Linux, GCC

A from-scratch, C-based virtual file system that simulates the core storage structures of a real OS: superblocks, inodes, and directory entries. Two command-line utilities, mkfs_builder and mkfs_adder, initialize virtual disk images and insert files into them, with CRC32 checksums used throughout to guarantee file system integrity. This project is also what inspired the "blueprint / schematic" look of this website.
